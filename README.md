Indian Election Assistant 🇮🇳🗳️

An interactive and visually engaging web application designed to simplify the understanding of the Indian election system. This project provides educational content about the election process, voter registration, democratic institutions, and includes an AI-inspired chat assistant interface for user interaction.

🚀 Features
✨ Modern and responsive UI
🗳️ Step-by-step Indian election process timeline
🎴 Interactive democracy flashcards
🏛️ Information about Lok Sabha, Rajya Sabha & State Assemblies
📝 Voter registration form guidance
🤖 AI Chat Assistant widget interface
📱 Mobile-friendly design
🎨 Smooth animations and clean layout
